# insta clone that opens with Live Server —
Pages & Features:

🔐 Login screen — type any username and hit Log In
🏠 Feed — posts with like, comment, save, and double-tap like
📖 Stories — animated story viewer with progress bar (auto-closes in 5s)
🔍 Search — live search dropdown in navbar
🧭 Explore — photo grid with hover overlays
🔔 Notifications — activity feed
💬 Messages — DM layout with conversation list
👤 Profile — stats, bio, photo grid, edit profile with avatar change
📤 Upload — pick a real photo from your computer and post it
How to open:

Download all 3 files keeping the folder structure (instagram/index.html, instagram/css/style.css, instagram/js/app.js)
Open the instagram folder in VS Code
Right-click index.html → "Open with Live Server"
Done! ✅
